JavaTM Accessibility Utilities

The Java Accessibility Utilities help assistive technologies take advantage of the Java Accessibility API. The utilities monitor component-related events. They also help assistive technologies get additional information about the GUI, such as the current position of the mouse, or the window that currently has focus.
