About the Examples

The Java Accessibility Utilities release includes example applications that use the Java Accessibility Utilities API (JAUAPI). The source files for the examples are under the examples directory in your Java Accessibility Utilities installation directory. Compiled versions are in the jaccess-examples.jar file included in this release. Among the examples are the following:
  • Ferret

    Ferret uses the JAUAPI to examine accessible information about the objects in the Java virtual machine. It allows the user to select different methods for selecting the object to be examined: by following focus, by following the mouse, by following the text caret, or by pressing the F1 button when the mouse pointer is over an object. Once an object has been selected, Ferret displays the accessible information for the object.
     

  • Monkey

    Like a monkey, Monkey "swings" through the component trees in a Java virtual machine and presents the hierarchy in two different ways. The first is the actual Component hierarchy and the second is the hierarchy viewed as Accessible objects. Also, if the user clicks the right mouse button over a tree node in Monkey, Monkey presents the user with a popup menu containing options for manipulating the object.
     

  • AccessibilityMonitor

    AccessibilityMonitor monitors all accessible property change events on all of the non-transient objects in the virtual machine. Also, if the user clicks the right mouse over the events presented in the table provided by AccessibilityMonitor, AccessibilityMonitor presents the user with a popup menu containing options for manipulating the associated object.

Before Using the Examples

 Before using the examples, you must do all of the following:
  • Download and install the Java 2 Platform SDK v1.4
     
  • Download and install the Java Accessibility Utilities v1.4
     
  • Copy jaccess.jar and jaccess-examples.jar (included in the Java Accessibility Utilities download) to the jre/lib/ext directory under your Java 2 SDK installation directory
     
  • Create or modify the jre/lib/accessibility.properties file under your Java 2 SDK installation directory to include a line such as the following:

    assistive_technologies=Ferret,Monkey,AccessibilityMonitor

    The exact line content depends on the assistive technologies you want to use.
