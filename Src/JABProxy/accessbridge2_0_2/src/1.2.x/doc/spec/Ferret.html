Java Accessibility Utilities

Starting Ferret

If you have completed the actions listed in Before Using the Examples, Ferret starts automatically any time you start a Java application.

Using Ferret

To use Ferret's capabilities, select an option from the Settings menu. When Ferret starts, no options are in use until you select one. The following table describes the options.
Option Description
Track Mouse If selected, this option causes Ferret to display accessible information for the object under the mouse pointer each time the mouse stops moving.
Track Focus If selected, this option causes Ferret to display accessible information for the object that currently has the focus, each time the focus changes.
Track Caret If selected, this option causes Ferret to display accessible information for the text object containing the text caret each time the caret moves.
Update with F1 Key If selected, this option causes Ferret to display accessible information for the object under the mouse pointer each time you press the F1 key.
