JavaTM Access Bridge
for the Microsoft Windows Operating System

Version 2.0.1

  
The Java Access Bridge is a technology to expose the Java Accessibility API (JAAPI) in a Microsoft Windows DLL, making Java applications and applets that implement the JAAPI visible to assistive technologies on the Microsoft Windows platform.

This release is designed to work with the Java SDK versions 1.2.2 through 5. It is not compatible with JDK 1.1.8. For more information about using the Access Bridge with other versions of the Java platform, see the Compatibility Table.
